Beta v3.2.17
InboxBuffer is a lightweight tool that helps developers and testers validate email functionality without sending real emails in a safe, controlled environment.

Key Features

Open Source

Entire codebase is publicly available on GitHub.

ESMTP

Extended SMTP server, allowing it to handle modern email protocols with extended features like authentication, custom headers.

Easy-to-Use Interface

View HTML, plain text, and attachments in a clean dashboard.

Quick Integration

Works with any app by using standard SMTP settings.

Self-Hosted

Can be deployed on your own infrastructure (local machine, Docker, or server).

JSON API

Useful for API integrations or automation workflows.

Safe Testing

No risk of spamming real users during development.
